# AI Supply Chain Disruption Predictor

A hackathon-ready MVp for supply chain resilience.

## Architecture
- **Frontend**: React, TS, Tailwind, Zustand (Vite)
- **Backend**: Node, Express, TS
- **AI**: Abstracted mock integration via Gemini API
- **Routing**: Mock Dijkstra placeholder for disruption avoidance

## Setup
### Backend
1. `cd backend`
2. `npm install`
3. `npm run dev` (starts on port 3001)

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev` (starts on Vite's default port, usually 5173)

Enjoy the beautiful hackathon UI dashboard!
